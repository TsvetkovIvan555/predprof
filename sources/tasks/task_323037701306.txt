Идентификатор некоторого ресурса сети Интернет имеет следующий вид: http://www.ftp.ru/index.html. Какая часть этого идентификатора указывает на протокол, используемый для передачи ресурса? Выпишите нужную часть.
Протокол  — часть адреса, стоящая до двоеточия. 1. www  — world wide web. 2. ftp  — название сайта. 3. http  — протокол. 4. html  — расширение файла с интернет страницей.

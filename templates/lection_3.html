Динамическое Программирование для ЕГЭ

Динамическое программирование - это метод решения сложных задач путем их разбиения на более простые подзадачи. Ключевая идея - сохранение результатов решения подзадач для избежания повторных вычислений. Этот подход особенно эффективен для задач, где одни и те же подзадачи встречаются многократно.

В контексте ЕГЭ, динамическое программирование часто применяется в задачах на поиск оптимальных путей, подсчет количества вариантов и задачах с последовательностями.

Основные Принципы Динамического Программирования

  • Разбиение на подзадачи: Исходная задача разбивается на более мелкие подзадачи, которые имеют такую же структуру.
  • Сохранение результатов: Результаты решения подзадач сохраняются в памяти (чаще всего в массиве или таблице) для дальнейшего использования.
  • Использование сохраненных результатов: При необходимости решения подзадачи, ее результат берется из памяти, а не вычисляется заново.

Пример Задачи: Задача о Рюкзаке

Рассмотрим классическую задачу о рюкзаке, где есть предметы с определенными весами и стоимостями, и рюкзак с определенной вместимостью. Цель - максимизировать общую стоимость предметов в рюкзаке, не превышая его вместимость.

Пусть есть предметы:

Предмет Вес Стоимость
1 2 60
2 3 100
3 4 120

И вместимость рюкзака: 5.

Решение с использованием динамического программирования

Мы создадим таблицу dp, где dp[i][w] означает максимальную стоимость, которую можно получить, используя первые i предметов и имея вместимость рюкзака w.

Алгоритм:

  1. Инициализируем таблицу dp нулями.
  2. Для каждого предмета i и вместимости w:
    • Если вес текущего предмета больше текущей вместимости w, то dp[i][w] = dp[i-1][w] (не берем предмет).
    • Иначе, dp[i][w] = max(dp[i-1][w], dp[i-1][w-weight[i]] + value[i]) (берем или не берем текущий предмет и выбираем лучшее).

Вот заполненная таблица dp:

0
Предмет\Вместимость 0 0
1 0 0 60 60 60 60
2 0 0 60 100 160 160
3 0 0 60 100 160 180

Ответ: 180. Это максимальная стоимость предметов, которые можно поместить в рюкзак вместимостью 5.

Пример кода на Python


def knapsack(weights, values, capacity):
    n = len(weights)
    dp = [[0 for _ in range(capacity + 1)] for _ in range(n + 1)]

    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if weights[i-1] > w:
                dp[i][w] = dp[i-1][w]
            else:
                dp[i][w] = max(dp[i-1][w], dp[i-1][w-weights[i-1]] + values[i-1])
    return dp[n][capacity]

weights = [2, 3, 4]
values = [60, 100, 120]
capacity = 5
print(knapsack(weights, values, capacity)) # Output: 180

Другие типы задач

Кроме задачи о рюкзаке, динамическое программирование применяется в следующих типах задач:

  • Задача о наибольшей возрастающей подпоследовательности: Нахождение самой длинной возрастающей подпоследовательности в заданной последовательности чисел.
  • Задача о расстоянии Левенштейна: Нахождение минимального количества операций (вставка, удаление, замена) для преобразования одной строки в другую.
  • Задачи на графах: Нахождение кратчайших путей или количества путей между двумя вершинами.

Советы для ЕГЭ

Основные моменты:

  • Определите, можно ли разбить задачу на подзадачи.
  • Определите, как хранить результаты подзадач (массив, таблица).
  • Определите, как использовать результаты подзадач для решения исходной задачи.

На практике: На ЕГЭ важно правильно определить, что задача может быть решена с помощью динамического программирования. Попробуйте упростить задачу до подзадачи, чтобы было легче ее понять.
